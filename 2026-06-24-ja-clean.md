日本語ユーザーのために最適化された、Typora向けのクリーンなカスタムテーマです。

- **日本語に最適化されたタイポグラフィ**: 游ゴシックやヒラギノ角ゴシックをベースにした、読みやすいフォントスタック。
- **詰まり気味のスマートな行間**: 大量の文章を1スクロールで素早く読めるよう、行間やマージンを意図的に狭めに設計しています。
- **近接の原則に基づくレイアウト**: 見出しと本文の関連性が視覚的に分かりやすいよう、上下の余白バランスを調整済みです。
- **洗練されたコードブロック**: デジタル・アートな雰囲気を醸し出す `Source Code Pro` を採用し、コードの視認性を高めました。

*Windows 11で設計およびテスト済みです。完全なテストは行っていませんが、macOS/Linuxでも動作するはずです。ただし、macOS特有のシームレスウィンドウスタイルなどの専用スタイルは含まれていません。*

*注記: このテーマは crisp-gothic.css をベースにしており、日本語の読みやすさのために最適化されています。*


A clean custom theme for Typora, optimized for Japanese users.

- **Typography optimized for Japanese**: A highly readable font stack based on Yu Gothic and Hiragino Sans.
- **Smart and tight line spacing**: Intentionally designed with tighter line spacing and margins to allow reading large amounts of text quickly with less scrolling.
- **Layout based on the law of proximity**: Adjusted top and bottom margin balances to visually clarify the relationship between headings and text.
- **Sophisticated code blocks**: Uses `Source Code Pro` to create a digital art vibe while enhancing the visibility of the code.

*Designed and tested on Windows 11. Not fully tested but should work for macOS/Linux. However, this theme may not contain platform-specific styles like macOS seamless window.*

*Note: This theme is based on the crisp-gothic.css theme structure, optimized for Japanese readability.*
